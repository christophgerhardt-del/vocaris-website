Automatisierung & eigener Webhook

Über Zapier / Make / n8n erreichst du jedes Tool (auch CRMs, die oben nicht gelistet sind) — oder gib einen eigenen Endpunkt an.

In Zapier/Make „Webhook → Catch Hook" anlegen, URL hier einfügen. JSON-POST, Echtheit per Header X-Vocaris-Signature: sha256=HMAC(Body).

Beispiel-Payload anzeigen { "event": "message.captured", "occurred_at": "2026-06-18T12:00:00.000Z", "data": { "caller_name": "Max Mustermann", "callback_number": "+49 170 1234567", "summary": "Rückruf gewünscht zu Angebot", "category": "Angebot", "urgency": "mittel" } }

„Speichern" sichert Ereignisse, Benachrichtigungen & Webhook. CRM-Verbindungen werden direkt beim „CRM verbinden" gespeichert. „Test senden" prüft alle aktiven Ziele (legt ggf. einen Test-Lead an).

Neue Nummer hinzufügen

Such dir eine freie Rufnummer und ordne sie einer Abteilung zu. Sie wird automatisch mit deinem Assistenten verbunden.

Hinweis: Deutsche Rufnummern erfordern bei der Bestellung einen Adressnachweis (Regulatory-Prüfung). Die Nummer wird automatisch aktiv, sobald die Prüfung durch ist.
